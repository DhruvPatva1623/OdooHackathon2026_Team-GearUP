# Code of Conduct

This project is dedicated to providing a welcoming environment for all contributors. All participants are expected to:

- Be respectful and considerate.
- Use inclusive language.
- Avoid harassment, discrimination, or abusive behavior.
- Accept constructive feedback gracefully.

## Expected Behavior

- Collaborate constructively with maintainers and contributors.
- Help keep discussions focused and productive.
- Report violations promptly to project maintainers.

## Unacceptable Behavior

- Harassment or discrimination of any kind.
- Personal attacks, insults, or threats.
- Sharing inappropriate content.
- Derailing conversations or disrespecting other contributors.

## Enforcement

Project maintainers may take action for any behavior that violates this code of conduct, including:

- issue warnings
- removing offensive content
- limiting or blocking contributors
- closing or rejecting contributions

If you experience or witness unacceptable behavior, please contact the repository maintainers through GitHub or the project communication channels.
